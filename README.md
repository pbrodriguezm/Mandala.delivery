## Empezar
`$ npm install`

## Run server
Run `ng serve` dev en server. Navigar a `http://localhost:4200/`. .
`$ ng serve -env=dev`


##Compile

`$ ng run build --prod`

## Running unit tests

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).
